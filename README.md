# PortfolioHUB

Repositório desenvolvido para armazenar projetos acadêmicos e pessoais do curso de Ciência de Dados e Machine Learning no CEUB.

## Objetivo
Organizar projetos utilizando Git, GitHub e versionamento de código, criando um portfólio profissional para apresentação acadêmica e profissional.

## Estrutura do Repositório

### banco_dados
Projetos, exercícios e materiais relacionados à disciplina de Banco de Dados.

### estudos
Anotações, exercícios e estudos de programação e ciência de dados.

### projetos
Projetos desenvolvidos utilizando HTML, CSS, Python e GitHub.

## Tecnologias Utilizadas
- Git
- GitHub
- HTML
- CSS
- Python

## GitHub Pages
Este projeto possui integração com GitHub Pages para publicação online.
